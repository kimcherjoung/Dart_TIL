#Dart 
## 기본적인 데이터 타입
```
void main() {
  String str = 'string';
  int in = 12;
  double dou = 1.2;
  bool bo = true;
  num n = 12;
  num n = 1.2;
}
```
위에서 순서대로 문자열인 String,숫자형인 Int,소수형인 Double,true,false인 boolean,그리고<br>
dart는 객체지향언어중에서도 심한편이라서 거의 object,와 class로 이루어져 잇는데 num은 int,double의 부모 클래스이기 때문에 num은 정수,소수 둘다 지정해줄 수 있다
